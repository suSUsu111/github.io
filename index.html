SimpleDateFormat问题

发表于 2017-12-15

在使用SimpleDateFormat时报错：
Didn’t find class “android.icu.text.SimpleDateFormat”
解决：
将import android.icu.text.SimpleDateFormat替换为java.text.SimpleDateFormat;
原因：
The android.icu.text.SimpleDateFormat is the ICU replacement for java.util.text.SimpleDateFormat;but it’s only available starting from API 24.
